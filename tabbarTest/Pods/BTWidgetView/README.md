# BTWidgetView

这是一个开发中积累的```UI```组件库，帮助你在开发中快速实现一些特定样式的布局效果，提高开发的效率。

它会基于[BTHelp](https://github.com/StoneMover/BTHelp)组件库中的一些工具类来完成一些功能。

由于该库中的```UI```组件较多，以下将采取分类的方式展示他们使用说明。

## 如何使用

```
pod 'BTWidgetView'
```
